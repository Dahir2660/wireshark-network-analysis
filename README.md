# Wireshark Network Traffic Analysis Lab

## Overview
This project demonstrates basic network traffic analysis and packet inspection techniques using Wireshark in a controlled lab environment.

## Objectives
- Capture network traffic
- Analyze packets and protocols
- Understand network communication
- Identify common protocols
- Practice basic traffic investigation

## Tools Used
- Wireshark
- Kali Linux
- Linux networking tools
- OpenSSH
- Nmap

## Skills Demonstrated
- Packet analysis
- Network troubleshooting
- Protocol analysis
- Traffic monitoring
- Cybersecurity investigation
- Network visibility

## Lab Activities
### Packet Capture
Captured live network traffic and analyzed packet flow between systems in a lab environment.

### Protocol Analysis
Observed and analyzed common protocols including:
- TCP
- UDP
- DNS
- HTTP
- SSH

### SSH Traffic Observation
Analyzed encrypted SSH traffic and studied authentication-related network activity.

## Key Takeaways
This lab improved my understanding of network communication, packet inspection, protocol behavior, and traffic analysis techniques commonly used in SOC and cybersecurity environments.
